abstract suspend fun getOG(url: String): Result<GetOGResponse>

Fetches Open Graph metadata for a URL.

This is useful for showing URL enrichment to users before an activity or comment is sent. You can then persist the OG data in attachments when sending the activity or comment (set skipEnrichUrl to avoid duplicate previews).

Return

A Result containing the GetOGResponse if successful, or an error if the request fails. Use GetOGResponse.toAttachment to convert the response to an io.getstream.feeds.android.network.models.Attachment.

Parameters

url

The URL to fetch OG metadata for.
