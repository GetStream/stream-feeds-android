CommentReactionsQuery

data class CommentReactionsQuery(val commentId: String, val filter: CommentReactionsFilter? = null, val limit: Int? = null, val next: String? = null, val previous: String? = null, val sort: List<CommentReactionsSort>? = null)

A query configuration for fetching reactions for a specific comment.

This struct defines the parameters used to fetch reactions for a comment, including pagination settings, sorting options, and filtering capabilities.

Features

  • Pagination: Supports next and previous cursors for efficient pagination

  • Sorting: Configurable sorting options for reaction ordering

  • Filtering: Supports filtering by reaction type, user ID, and creation date

Parameters

commentId

The unique identifier of the comment to fetch reactions for.

filter

Optional filter criteria to apply to the reactions. Use this to narrow down results based on specific criteria. See CommentReactionsFilterField for available filter fields and their supported operators.

limit

The maximum number of reactions to return in a single request. If not specified, the API will use its default limit.

next

Pagination cursor for fetching the next page of results. This is typically provided in the response of a previous request.

previous

Pagination cursor for fetching the previous page of results. This is typically provided in the response of a previous request.

sort

Array of sorting criteria to apply to the comment reactions. If not specified, the API will use its default sorting.
