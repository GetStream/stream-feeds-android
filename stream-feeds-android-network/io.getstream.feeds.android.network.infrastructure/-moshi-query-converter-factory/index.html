MoshiQueryConverterFactory

class MoshiQueryConverterFactory(moshi: Moshi) : Converter.Factory

A Retrofit Converter.Factory that JSON-serializes complex objects used as @Query parameters.

Retrofit's default behavior for @Query is to call toString() on the value, which produces a Kotlin data class string representation instead of JSON. This factory intercepts non-primitive types and serializes them to JSON via Moshi.
